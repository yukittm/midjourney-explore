# Exploration Map

Plan for exploring NEW subject coverage for the IG base runway (~11 pieces). The **style/signature is
fixed** (flowing-color bands/grain via `--sref` + a photoreal anchor + the bridge); we vary the
**subject**, and within each subject the **sub-approaches** (species/form · composition/scale · angle ·
wave-mode · light/time · setting · extra elements). Goal: a **diverse but cohesive** grid — avoid the
over-explored sameness.

## Per-area method
Frame the exploration BEFORE generating (decide what varies vs what holds) → **cross-review with independent
agents** → generate → iterate on what lands. **Do NOT grid-lock**: the levers below are an optional palette to
*diverge* across a batch, not a factorial matrix to fill (locked patterns = the sameness failure; the user's
standing philosophy is minimal style constants + deliberate divergence).

**Lever palette** (pick a few, vary them deliberately):
- subject-type / form · composition & shot-distance · angle (incl. top-down / worm's-eye / dutch)
- **lens / optical treatment** (backlit translucency · macro-bokeh · telephoto compression · refraction)
- **wave-mode** (the 7 above) — the color's *manifestation*
- **★ bridge-phenomenon = WHERE the color physically lives** (sky / ground · terrain / water / **on the
  subject itself — incl. its moving parts, mode 7**) — LOAD-BEARING; vary it across a batch or color attaches
  generically → sameness
- light-behavior · setting · subject-state

## Parked exploration candidate (NOT a fixed base — one of many things to try)
**"Reality-defying constructs" lane:** scenes/objects that deliberately break the structural laws of the
real world (geometry, perspective, gravity, topology, continuity, physics, scale, dimensionality), built so
**the parts read photoreal but the whole is impossible** ("believable parts, impossible whole"). Pure-abstract
impossibility failed in-app; the viable form is **photoreal-grounded impossibility**. Specific devices
(folding / multi-gravity space, topological twists, impossible geometry, anti-physics objects) are just
**instances of the principle**, not the definition. **Status: VALIDATED 2026-06-27, narrowed 2026-07-02.** The
viable trigger found in-app: **remove the floor** (no ground / horizon / landscape → the form floats in a void
and can twist), and let a **single continuous surface carry figures at multiple orientations** (some upright,
some inverted). A 4-probe batch (engulfment / void-float / wave-as-sky / color-vortex, 2026-07-01) came back
mostly mediocre — **engulfment is the promoted device** (now Active lane A); the other three stay parked, not dead.
