**CONTROLLED FINDING (primary lever = the simile, not the params).** Two MJ renders, IDENTICAL params (`--raw --sw 200 --hd --ar 4:5`, `--s` default/unset, same image `--sref`):
- Aqueduct, bridge simile "...like real WIND-COMBED DUNES" → rendered FINE, tight, high-frequency combed contour striping (the thin wavelength the user DISLIKES).
- Concrete monolith, bridge simile "sediment bands sweeping like real ZHANGYE DANXIA rock" → rendered BROAD, wide, low-frequency sweeping bands (the signature "なみなみ").
Same params → opposite wave character. So within the raw/sw200 regime the FINE↔BROAD character tracks the NAMED real-world phenomenon in the "like real {X}" simile. --raw/--s is at most a secondary modulator, NOT the character lever.

**WHY (physics of each phenomenon predicts the render):**
- Sand ripples/dunes = high-frequency granular bedforms, wavelength ≤~24cm, straight symmetric repeated crests → MJ reads "dunes/ripples/wind-combed/corrugated" as MANY fine parallel lines. AVOID. (arxiv/geology sources on aeolian ripple wavelength.)
- Ocean wind-chop = chaotic short-wavelength ripples stacking; ground/groundswell = organized LONG-wavelength (12–15s period), smooth, far-apart, sorted, few large swells → "slow ocean groundswell" yields BROAD. (scienceofsurfing / surfertoday / wikipedia Swell.)
- Zhangye Danxia = "expansive layered hills... broad geological cross-sections," strata "resembling countless bolts of colorful silk," continuous undulating bands, 30+ colors → BROAD sweeping bands. (whc.unesco.org tentativelists/6841, travelchengdu, chinadragontravel.)
- Molten/poured glass: SLOW controlled pour → "broader, flowing color bands"; fast/high-speed → "fine streaks" (injection-molding marbling source). So specify SLOW/molten, not "swirl."

**VETTED BRIDGE VOCABULARY:**
- BROAD / signature (USE): "sediment bands sweeping like real Zhangye Danxia strata"; "slow ocean groundswell, long-period smooth swells"; "slow molten poured glass, broad flowing bands"; "banded agate / onyx cross-section, few wide bands"; "wind-sculpted Navajo sandstone (The Wave, Arizona), sweeping cross-bedded swirls"; "poured-paint marbling, wide slow ribbons."
- FINE / avoid: dunes, sand ripples, ripples, wind-combed, combed, corrugated, fluted, zebra striping, wood-grain, fingerprint whorls.

**LANGUAGE KNOBS for amplitude/wavelength (add explicit count + size):** "a FEW (3–5) broad low-frequency swells," "wide wavelength," "large-amplitude sculptural swells," "slow rolling," "molten." REMOVE frequency-implying words: "rippling," "undulating" (mid), "curling," "combed," "wind-combed."
