**Cliché ranking (most generic → most ownable):**
- INSTANT STOCK (avoid as primary): nebula (purple/teal cloud) — the single most saturated AI/stock subject; spiral galaxy face-on; milky-way star-field; cosmic dust clouds. These ARE "AI space wallpaper."
- MEDIUM (common but workable with restraint): aurora (over-saturated rainbow curtains = cliché; subtle banded over minimal horizon = ownable — flowing-color asset), ringed Saturn-planet (3D-render look is the cliché), supernova/explosion burst, comet.
- RARER / MORE OWNABLE: black-hole accretion disk (harder for AI to cliché-ify, reads intentional), solar eclipse/corona (graphic ring-of-fire = strong negative-space subject), meteor shower (as sparse streaks over minimal land, not busy). Rarity comes more from TREATMENT than subject.

**Generic-AI-space tropes (the tells):** hyper-saturated purple+teal default palette; symmetrical/lens-flare nebula with central bloom; dead-center symmetry; fake star bokeh + over-dense busy star-field (AI over-detailing — "gets worse the longer you look," repeating patterns where chaos should be); glossy 3D-render planet (CGI sheen, octane look); everything in focus + uniform glow; no foreground/no scale anchor (pure backdrop = wallpaper by definition). AI-tell sources: roblaughter Medium 14-signs, skillshare/artsmart guides.

**Anti-cliché composition (turns wallpaper into editorial/fine-art):**
- Small ground element / lone figure or tree under epic phenomenon — the project signature. VALIDATED by Storm Thorgerson (small figures vs vast epic landscapes, "experimental treatment of scale") + astrophoto rule (sky 2/3, foreground 1/3, single anchor: lone tree/rock/figure/ruin). Best single carrier of the signature.
- Surreal oversized planet/galaxy/moon looming over minimal landscape — Mariano Peccinetti / Thorgerson collage lineage (giant celestial object over sparse desert/beach). Strong ownable route.
- Cosmic color reflected in water / salt flat / mirror — Murray Fredericks Lake Eyre "saltscapes" (color + abstraction + minimalism, near-perfect reflection). Doubles the flowing-color, adds restraint.
- Extreme negative space + restraint — empty dark field, phenomenon small/off-center, NOT filling frame. Rule-of-thirds placement, not dead center.

**"Flowing color" PAINTERLY vs render/photo:** aurora/nebula/galaxy-arm are literally flowing light. To read PAINTERLY not stock-photo/3D-render: `--style raw` + LOW `--s` to kill CGI gloss (per [[mj-style-raw]]); AVOID render keywords (octane/unreal/rtx/8k → sterile CGI per [[mj-optical-refraction-spectral]]); use medium/analog cues (gouache/oil/ink-in-water/painted, film grain, long-exposure) and motion cues (flowing, drifting, banded) so color reads as brushed bands not photographic gas. Iridescence/oil-slick keywords give continuous flowing color fields (cliché-safe) vs banded rainbow. OPINION: painterly + restrained beats photoreal-nebula for ownability.

**Palette / ownability:** default AI cosmic = purple+teal (stock-saturated — Getty/iStock/Dreamstime confirm). A CONTROLLED, non-default palette is the strongest ownability lever (consistent signature across a feed). Carry color via `--sref` + `--sw` and explicit color language, NOT via high `--stylize` (high stylize re-averages toward MJ default + washes sref — per [[mj-stylize-chaos-hd-params]]). Pick a chord that is NOT purple/teal nebula (e.g., the project's existing flowing-color bands) and hold it.
